Readme for the OASIS committee-note plug-in
Original creation: KJE 30 Jan 2019
Updated 20 Feb 2019 (font and OT version change)
--------------------------------------------------------------------

This plug-in runs on DITA-0T 3.2.1 and uses Antenna House Formatter. 
It requires a system with the Liberation Sans font installed.
The transformation that it defines is "oasis-pdf-committeeNote".
